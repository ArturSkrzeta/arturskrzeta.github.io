Linear Regression with Pure Math

Intro

Regression analysis is a predictive method that allows to investigate the relationship between y (dependent variable) and x (independent value). In regressing we want to predict continueous values as opposed to classification model where we want to predict the class of a case i.e. label or 0/1 category.

Regression application:

  • Determining the strength of ralationships betwenn variables - of how x values effects y values:
    - e.g. relationship between employee's age and income, earnings and marketing spend.
  • Forecasting impact or effect on y values by changing one or more x values:
    - e.g. how much income increases when 10 000 USD invested in the specific business field.
  • Trend and future values forcasting:
    - estimating the sales in the future when trend is stable in time.

Demo

Fitting linear function to data points:

  1. Generating random data.
  2. Splitting data into training and testing subset.
  3. Training model with training subset - x training features and y training values:
    - training model depends on fitting linear function as much as possible to the data points
  4. Testing model with testing subset:
    - putting x testing features through a model
    - model gives y predicted values
    - we compare y predicted values dividing them by y testing values getting model accuracy
  5. Once accuracy satisfies, we can put through model new sample features model has never seen.
  6. Model follows linear functon and adjusts y values for given x sample features.
  • As above for 5 new features as an input, model gives back 5 corresponding values due to linear function.
